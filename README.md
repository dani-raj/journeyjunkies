# Journey Junkies

A responsive, single-page travel website built with **HTML, CSS and vanilla JavaScript**.

This project was originally created several years ago as a beginner site-building exercise and has recently been **lightly refactored and cleaned up** to reflect more solid fundamentals and better UX practices.

## What this project demonstrates

- Responsive layout (mobile, tablet, desktop)
- Semantic HTML structure
- CSS layout techniques (Flexbox, Grid, transitions)
- Vanilla JavaScript DOM manipulation
- Simple form validation and user feedback
- Basic state persistence with localStorage
- Mobile navigation and interactive image gallery

## Tech stack

- HTML5  
- CSS3  
- Vanilla JavaScript  
- No frameworks, no build tools

## Notes

This is not a production application, but a **site-building focused project** intended to demonstrate clean structure, basic interactivity, and attention to detail when working without frameworks.

The focus was on improving clarity, maintainability and mobile behavior without introducing unnecessary complexity.
